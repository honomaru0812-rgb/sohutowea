
## ルール
1. **自分のファイルだけ編集する**（他の人のファイルは触らない）
2. 新しいCSSクラスが必要な場合は、デザイン担当に依頼する
3. HTMLに要素を追加したい場合は、JavaScriptで動的に作る
4. 困ったらまず `window.App` の中身をコンソールで確認する
5. `console.log()` でデバッグする

## 開発の始め方
1. このフォルダをチーム全員に共有する
2. 各自、自分のファイルを編集する
3. `index.html` をブラウザで開いて動作確認
4. 完成したファイルをあなたに渡す → 統合
